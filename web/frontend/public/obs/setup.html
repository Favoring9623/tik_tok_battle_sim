OBS Setup Instructions
1
Add Browser Source

In OBS, click the "+" button in Sources and select "Browser"

2
Paste the URL

Copy the overlay URL above and paste it into the "URL" field

3
Set Dimensions

Full Overlay: 1920 x 1080
Score Widget: 400 x 100 (or 150 for vertical)
Gift Alerts: 400 x 600

4
Enable Transparency

Scroll down and check "Custom CSS" is empty, then the transparent background will work automatically

5
Position the Overlay

Drag and resize the overlay in OBS to your preferred position

6
Start a Battle!

Run a battle and the overlay will update in real-time via WebSocket

Tip: Make sure the battle server is running on the same machine or network. The overlay connects via Socket.IO for real-time updates.
